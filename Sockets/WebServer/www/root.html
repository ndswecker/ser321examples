You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns the multiplication of two numbers
  • /binary -- returns input query in binary
  • /DNDcoinage -- converts dnd coinage into copper currency

File Structure in www (you can use /file/www/FILENAME):

multiply (/multiply?num1=NUMBER&num2=NUMBER)

  • only entering one value squares the number
  • not entering a valid number throws and error
